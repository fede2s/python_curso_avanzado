Tabla con datos:

En el constructor del controlador, instancio una tabla de base de datos donde en su constructor:

  1. Primero verifico si existe la base y si no la creo

  2. Después verifico si la tabla está creada en la base de datos y si no la creo

Después, en el método inicializar del controlador y llamando métodos de la vista y métodos de la tabla de la base de datos:

  1. Defino las funciones ABM y Alerta para manejar la lógica entre la vista y el modelo (tabla en base de datos)

  2. Consulto datos de la tabla a la base de datos

  3. Consulto los títulos de las columnas de la tabla en la base de datos

  4. Construyo la tabla en Flet asignándole los títulos a cada columna y los datos a cada fila
